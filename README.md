# Dots

My Ansible-based dotfiles repository. Built pretty nicely if I do say so
myself.

**Supported Linux Distributions:** Fedora

## Prerequisites

The prerequisites to use this set of Dotfiles is as follows:

- Linux/WSL
  - sudo permissions
- Ansible
- [Git](https://git-scm.com)

## Installation

```sh
# Run the bootstrapper playbook (including required env vars)
$ ./bootstrap.sh
```
